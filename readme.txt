# VerzaAI

## Installation
1. Clone the repository: `git clone https://github.com/yourusername/yourrepository.git`
2. Install the required dependencies:

3. Create API key files in the `API-Keys` directory:
   - `pinecones.txt`
   - `groq.txt`
4. Update the API keys in the `start.py` file.

## Usage
1. Run the program: `python start.py`
2. Enter text prompts when prompted.
3. Type "exit" to end the program.

## Additional Notes
- Ensure Python 3.x is installed.
- Make sure to handle any errors that may occur during execution.


All conversations are stored in a Pinecones Vector Store.
